Sobre mí

Pienso en productos, no solo en código. Llevo una idea desde la arquitectura hasta el usuario final y me hago responsable de que funcione, sea rápida y se pueda mantener.

Mi base es el desarrollo móvil (Flutter, .NET MAUI, Xamarin, React Native) y el ecosistema .NET en escritorio y backend. Trabajo con arquitectura limpia y separación por capas, diseños offline-first donde el dominio no depende de la base de datos, y sincronización confiable con manejo de conflictos.

He construido desde un SaaS multi-tenant con pagos hasta software empresarial con impresión Bluetooth para fuerza de ventas en campo. Aprendo rápido lo que el proyecto necesita y entrego cosas que se usan en el día a día.

Cómo trabajo

Principios que aplico en cada proyecto, sin importar el stack.

Arquitectura limpia

Capas separadas y dominio independiente de la base de datos. Fácil de probar y de cambiar.

Offline-first y datos confiables

✨

UX antes que complejidad

Interfaces claras y rápidas. Menos fricción para el usuario, menos soporte para el negocio.

📈

Código pensado para crecer: multi-tenant, reutilizable y con monitoreo de errores.

Acelero con IA para entregar más rápido, pero reviso y decido yo. Calidad sobre velocidad ciega.
