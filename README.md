# cecaza.github.io

Portafolio personal de **Cesar Carvajal** — desarrollador de apps móviles (Flutter) y web (Next.js + Supabase).

🌐 **En vivo:** https://cecaza.github.io

## Qué hay aquí

Sitio estático (HTML/CSS, sin build) servido con **GitHub Pages**.

- `index.html` — portafolio: sobre mí, skills, proyectos e ideas.
- `dulce-espera/` — página de la app **Dulce Espera** (con capturas).
- `turno/` — página del proyecto **Turno** (agenda con seña).
- `cinexa/` — página del proyecto **Cinexa** (catálogo de cine).
- `privacidad.html` — copia fuente de la política de privacidad.

## Publicar / actualizar

Es estático: edita el HTML y haz `git push`. GitHub Pages se sirve desde la rama `main` (raíz).
Para activarlo la primera vez: **Settings → Pages → Branch: `main` / root → Save**.
